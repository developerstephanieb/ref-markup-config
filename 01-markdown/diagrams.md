### 3. Pie Charts

Simple percentage visuals for showing proportional data. Pie charts start with `pie` followed by an optional title.

**Example:**
````markdown
```mermaid
pie title Project Status
    "Completed" : 70
    "In Progress" : 20
    "Blocked" : 10
```
````

**Rendered:**
```mermaid
pie title Project Status
    "Completed" : 70
    "In Progress" : 20
    "Blocked" : 10
```
